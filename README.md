# AI Planning Assistant — Digital Colleague

MBA Thesis Proof-of-Concept | Macromedia University Munich

An AI-powered planning assistant that reads, cleans, and analyses
any CSV/Excel dataset, answers natural-language questions about the
data, and transparently shows how each answer is calculated.

## Research Context
- RQ1: What builds manager trust in AI recommendations?
- RQ2: Does AI-assisted planning improve efficiency?

## Run locally
\`\`\`
pip install -r requirements.txt
streamlit run app_thesis.py
\`\`\`

Sample dataset: Walmart M5 Demand Forecasting (Kaggle, public)
